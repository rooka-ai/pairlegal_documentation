# Fehlerbehebung

## Office 365 Berechtigungen

Wenn Sie den Abschnitt `Admin` in Ihrem Microsoft 365-Konto nicht sehen, bedeutet dies,
dass Sie nicht über ausreichende Rechte verfügen. Sehen Sie sich diese [Microsoft-Seite](https://learn.microsoft.com/en-us/microsoft-365/admin/manage/centralized-deployment-of-add-ins?view=o365-worldwide)
an, um einen vollständigen Überblick über die Anforderungen für das Hinzufügen von
MS Add-in zu integrierten Anwendungen zu erhalten.

## Office 365 Lizenz

Nur Business- und Enterprise-Lizenzen unterstützen das Hinzufügen von MS Add-in zu
integrierten Anwendungen. Sehen Sie sich diese [Microsoft-Seite](https://learn.microsoft.com/en-us/microsoft-365/admin/manage/centralized-deployment-of-add-ins?view=o365-worldwide)
an, um die vollständigen Anforderungen für das Hinzufügen von MS Add-in zu integrierten
Anwendungen zu erhalten.

## Add-ins-Symbol ist nicht sichtbar

Es kann vorkommen, dass Ihre MS Word-Einstellungen die Schaltfläche `Add-ins` auf der
Registerkarte `Start` in MS Word nicht anzeigen.

<Image img={home} />

### Schritte

1. Öffnen Sie ein beliebiges Dokument in MS Word und klicken Sie auf `Datei`
   <Image img={file} style={{ width: 600 }} />
2. Dadurch wird ein Seitenmenü geöffnet, wählen Sie `Add-ins abrufen`
   <Image img={addins} style={{ width: 200 }} />

<br />

Wenn Sie dieses Symbol auch nicht sehen, folgen Sie diesen Schritten:

1. Klicken Sie erneut auf `Datei`
2. Klicken Sie unten im Seitenmenü auf `Optionen`
3. Ein neues Fenster wird geöffnet
4. Klicken Sie auf `Menüband anpassen`
5. Fügen Sie Add-ins zur Registerkarte `Start` hinzu
