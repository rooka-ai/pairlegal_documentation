# Installation

Derzeit gibt es zwei Optionen, um das Plugin in MS Word zu installieren.

## Individuelle Installation über Microsoft AppSource

Das Plugin kann direkt in MS Word über Microsoft AppSource heruntergeladen werden.

### Schritte

**Gehen Sie zur Registerkarte `Start` und klicken Sie auf `Add-ins`**

  <Image img={home} />

**Klicken Sie auf `Weitere Add-ins`**

  <Image img={moreaddins} style={{ width: 200 }} />

**Klicken Sie auf `Store` und suchen Sie nach `Single Draft`**

  <Image img={search} style={{ width: 400 }} />

**Klicken Sie auf die Schaltfläche `Hinzufügen`**

  <Image img={add} style={{ width: 400 }} />

**Das Add-In wird unter `Meine Add-Ins` hinzugefügt**

  <Image img={myaddins} style={{ width: 400 }} />

### Mögliche Probleme während der Installation

Falls während der Installation ein Problem auftritt, besuchen Sie bitte diesen Link:
[Fehlerbehebungsleitfaden](troubleshooting.mdx).

## Masseninstallation über Office 365-Administration

Wenn Sie eine Office 365 Business- oder Enterprise-Lizenz haben und das Plugin für 
mehrere Benutzer hochladen möchten, ist dies über die Office 365-Administration möglich.
[Klicken Sie hier für die Installationsanleitung](install365.mdx)
