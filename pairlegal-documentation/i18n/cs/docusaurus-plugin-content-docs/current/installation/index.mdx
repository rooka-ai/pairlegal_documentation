# Instalace

Současně nabízíme dvě možnosti instalace doplňku do MS Word.

## Office 365

Pokud máte licenci Office 365 Business nebo Enterprise, je následující postup
preferovanou možností instalace doplňku.

- [Kroky instalace](install365.mdx)

## Manuální nahrání doplňku ("sideloading")

- [Windows](https://learn.microsoft.com/en-us/office/dev/add-ins/testing/create-a-network-shared-folder-catalog-for-task-pane-and-content-add-ins)
- [Mac](https://learn.microsoft.com/en-us/office/dev/add-ins/testing/sideload-an-office-add-in-on-mac)

:::tip Již brzy - Microsoft App Store

Uživatelsky pohodlnou možností instalace doplňku je instalace prostřednictvím Microsoft
App Store. Současně spolupracujeme s Microsoftem na tom, abychom mohli doplněk poskytnout
přímo v MS Word mezi ostatními nabízenými doplňky.
:::
