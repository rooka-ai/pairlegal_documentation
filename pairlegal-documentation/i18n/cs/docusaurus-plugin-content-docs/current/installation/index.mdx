# Instalace

Současně existují dvě možnosti, jak nainstalovat plugin do MS Word.

## Individuální instalace skrze Microsoft AppSource

Plugin je možné si přímo v MS Word stáhnout z Microsoft AppSource.

### Kroky

**Jděte na záložku `Domů` a klikněte na `Doplňky`**

  <Image img={home} />

**Klikněte na `Další doplňky`**

  <Image img={moreaddins} style={{ width: 200 }} />

**Klikněte na `Store` a vyhledejte `Single Draft`**

  <Image img={search} style={{ width: 400 }} />


**Klikněte na `Přidat` tlačítko**

  <Image img={add} style={{ width: 400 }} />

**Add-in je přidaný v `Moje Doplňky`**

  <Image img={myaddins} style={{ width: 400 }} />


### Možné problémy během instalace

V případě, že během instalace narazíte na problém, podívejte se prosím na tento odkaz
[Průvodce řešením problémů](troubleshooting.mdx).

## Hromadná instalace skrze administraci Office 365

Pokud máte licenci Office 365 Business nebo Enterprise a přejete si nahrát plugin
pro vícero uživatelů, je toto možné skrze administraci Office 365.
[Klikněte zde pro návod na instalaci](install365.mdx)
