# Individual Installation

The plugin can be added into user's add-ins directly in the Word using
Microsoft AppSource.

## Steps

1. **Go to `Home` tab in MS Word and click on `Add-ins`**

{" "}

<Image img={home} />

<br />
<br />

2. **Click on `More Add-ins`**

{" "}

<Image img={moreaddins} style={{ width: 200 }} />

<br />
<br />

3. **Click on `Store` and in the search field search for `Single Draft`**

{" "}

<Image img={search} style={{ width: 400 }} />

<br />
<br />

4. **Click on `Add` button**

{" "}

<Image img={add} style={{ width: 400 }} />

<br />
<br />

5. **The Add-in is visible in `My Add-Ins`**

{" "}

<Image img={myaddins} style={{ width: 400 }} />
