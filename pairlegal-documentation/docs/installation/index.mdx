# Installation

There are currently two options to install the add-in into MS Word.

## Office 365

If you have Office 365 Business or Enterprise license this is the preferred option
to install the add-in. [Click here to see installation steps](install365.mdx)

## Manual Sideloading

- [Windows](https://learn.microsoft.com/en-us/office/dev/add-ins/testing/create-a-network-shared-folder-catalog-for-task-pane-and-content-add-ins)
- [Mac](https://learn.microsoft.com/en-us/office/dev/add-ins/testing/sideload-an-office-add-in-on-mac)

:::tip Coming soon - Microsoft App Store

The most user friendly option to install the add-in is through Microsoft App Store.
We are currently cooperating with Microsoft to be able to provide the add-in
directly in MS Word among other public add-ins.
:::
