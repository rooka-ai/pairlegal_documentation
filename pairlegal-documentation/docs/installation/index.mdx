# Installation

There are currently 2 options to install the Add-in in MS Word.

## Office 365

If you have Office 365 Business or Enterprise license this is the most preferred option
to install the Add-in.

- [Installation steps](install365.mdx)

## Manual Sideloading

- [To install on Windows](https://learn.microsoft.com/en-us/office/dev/add-ins/testing/create-a-network-shared-folder-catalog-for-task-pane-and-content-add-ins)
- [To install on Mac](https://learn.microsoft.com/en-us/office/dev/add-ins/testing/sideload-an-office-add-in-on-mac)

## Coming soon

The most user friendly option to install the Add-in is through Microsoft App Store.
We are currently cooperating with Microsoft to be able to provide the Add-in
directly in MS Word among other public Add-ins.
