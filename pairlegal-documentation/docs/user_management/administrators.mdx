# Administrators

There are two types of user roles that any user can be given.

## Team Admin

How to become `Team Admin`? There are 2 ways:

1. Any user who creates subscription automatically becomes a `Team Admin` once an invoice is paid.
1. Any user who is already `Team Admin` can assign this role to any other user
   who does not have `Team Admin` role yet.

### Add Team Admin Role

Prerequisites:

1. The user who attempts to add this role to another user already has `Team Admin` role.
1. There are already users in at least one subscription of the company.

Steps:

1. Go to `Settings > Users management`
1. Click on `Your users`
1. From the list of users select the user and click on three dots in `Actions` column
1. Select `Make Team Admin`

<Image img={adminroles} />

### Team Admin Permissions

**Subscription**

- Cancel subscription paid by them.
- View all subscriptions in a company.
- View all users that belong to a subscription.
- Add user to a subscription.
- Remove user from a subscription.

**Roles**

- Add `Team Admin` role to any other user who does not have the `Team Admin` role yet.
- Add `Data Manager` role to any other user who does not have the `Data Manager` role yet.

**Data**

- Set/update/delete company secrets, e.g., DeepL API Key

## Data Manager

How to become a `Data Manager`? Ask any user from your company who has the
`Team Admin` role to give you the `Data Manager` role.

### Add Data Manager Role

Prerequisites:

1. The user who attempts to add this role to another user already has the `Team Admin` role.
1. There are already users in at least one subscription of the company.

Steps:

1. Go to `Settings > Users management`
1. Click on `Your users`
1. From the list of users select the user and click on three dots in `Actions` column
1. Select `Make Data Manager`

<Image img={adminroles} />

### Data Manager Permissions

**Data**

- Set/update/delete company settings, e.g., system prompt
