Troubleshooting

Office 365 Privileges​

If you cannot see Admin section in Microsoft 365 account it means you do not have sufficient rights. Please refer to this Microsoft webpage for full set of requirements for adding MS Add-in into Integrated Apps.

Office 365 License​

Only Business and Enterprise licenses support adding MS Add-in into Integrated Apps. Please refer to this Microsoft webpage for a full set of requirements for adding MS Add-in into Integrated Apps.

Add-ins Icon Not Visible​

It can happen that your MS Word customization does not have Add-ins button at the Home tab of the MS Word.

Steps​

  1. When you have any MS Word document open, click on File
  2. This will open side bar, select Get Add-ins

In case you do not see this icon either, please follow these steps:

  1. Click on File again
  2. At the bottom of the sidebar there is Options
  3. This will open new window
  4. Click on Customize Ribbon
  5. Add Add-ins to Home tab
